


for: es un ciclo que se ejecuta hasta que se cumpla una condicion especifica


`&&` (AND) : Esto evaluará ambas expresiones y devolverá verdadero si AMBAS expresiones son `true`. Si uno (o ambos) de ellos es falso, este operador devolverá `false`.



`||` (OR):  Determinará si una de las expresiones es `true`. Devolverá `true` si una (o ambas) de las expresiones es` true`. Devolverá `false` si AMBAS expresiones son` false`.





`!`  (NOT): el operador NOT devolverá el valor booleano opuesto de lo que se le pasa.
